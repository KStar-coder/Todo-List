
# Todo List

This application is made by exploring the application context under React and localStorage of the browser

## Run Locally

Clone the project

```bash
  https://github.com/KStar-coder/Todo-List.git
```

Go to the project directory

```bash
  cd Todo-List
```

Install dependencies

```bash
  npm i
```

Start the server

```bash
  npm run dev
```
